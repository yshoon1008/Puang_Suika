# CAU Puang Suika (중앙대 푸앙 수박게임)

객체지향프로그래밍 과제 프로젝트입니다.
물리 엔진(Box2D)과 그래픽 라이브러리(SFML)를 사용하여 제작했습니다.

## 게임 설명
똑같은 단계의 푸앙이를 합쳐서 더 큰 푸앙이로 진화시키는 게임입니다.
최종 목표인 **Blue Dragon**을 만들어보세요!

## 개발 환경
- **Language**: C++ 17
- **Library**: SFML 3.0, Box2D 2.4.1
- **Tool**: VS Code, CMake, vcpkg

## 조작 방법
- **마우스 이동**: 푸앙이 위치 조절
- **마우스 클릭**: 푸앙이 떨어뜨리기

## 실행 방법 (Build)
1. 이 저장소를 Clone 합니다.
2. `vcpkg`를 통해 SFML과 Box2D를 설치합니다.
3. CMake를 사용하여 빌드합니다.
4. `assets` 폴더가 실행 파일과 같은 경로(또는 상위)에 있어야 합니다.
